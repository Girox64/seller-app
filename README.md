#Seller APP

> The best Seller app of world, by the best team.

## Getting Startes

Install dependencies:

```shell
yarn
```

Start the dev API server:

```shell
yarn start:api
```

Start the dev server:

```shell
yarn start
```

Make your changes. 🍚

Commit your feature:

```shell
yarn commit
```

Ship it! 🚀
